Open Microscopy Environment - Tagged Image File Format (OME-TIFF) is a format standard for high-dimensional, metadata-rich microscopy images.

There are many tools available for image conversion (non-exhaustive list):

  • ome/bioformats (bfconvert CLI) Bio-Formats --> Bio-Formats
  • glencoesoftware/bioformats2raw (CLI) Bio-Formats --> intermediate Zarr structure compatible with the OME-Zarr specification
  • glencoesoftware/raw2ometiff (CLI) Intermediate Zarr structure (see bioformats2raw) --> OME-TIFF
  • glencoesoftware/NGFF-Converter (GUI) Graphical user interface for bioformats2raw & raw2ometiff
  • Euro-BioImaging/BatchConvert (CLI) Nextflow-based batch conversion tool using bfconvert & bioformats2raw
  • Python packages such as bioio (many formats), ngff-zarr (OME-Zarr), or ome-zarr-py
  • Any image viewer (GUI) supporting reading/writing using e.g. Bio-Formats, such as ImageJ/Fiji or QuPath
